Overview

agristoolkit is a Stata package designed to simplify the analysis and tabulation of survey data, with a focus on agricultural datasets.

Installation

Install directly from GitHub or your repository:

net install agristoolkit, from("https://your-url")

Main Features

  • Efficient tabulation of survey data
  • Support for complex survey design
  • Clean and reproducible workflows
